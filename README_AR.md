# 🏋️ نظام إدارة الصالة الرياضية Barcoda Gym

## المميزات الرئيسية ✨

### 1. إدارة الأعضاء 👥
- إضافة وتعديل وحذف أعضاء الصالة
- حفظ بيانات شاملة (الاسم، الهاتف، الصورة، إلخ)
- تتبع حالة العضوية (نشطة/معطلة)

### 2. نظام الباركود 📱
- إنشاء باركود فريد لكل عضو
- مسح الباركود عند دخول الصالة
- تتبع الحضور تلقائياً

### 3. إدارة الاشتراكات 💳
- خطط اشتراك مرنة
- تتبع تاريخ انتهاء الاشتراك
- تذكيرات تلقائية قبل الانتهاء
- تجديد الاشتراكات بسهولة

### 4. نظام الدفع 💰
- تسجيل عمليات الدفع
- دعم طرق دفع متعددة (نقد، كارت، تحويل)
- طباعة إيصالات الدفع

### 5. التنبيهات والإشعارات 🔔
- تنبيهات الاشتراكات منتهية الصلاحية
- إشعارات الدفع المعلق
- رسائل SMS/Email اختيارية

## المتطلبات 📋

- Node.js v14 أو أعلى
- MySQL 5.7 أو أعلى
- NPM أو Yarn

## التثبيت السريع 🚀

### 1. استنساخ المشروع
```bash
git clone https://github.com/muatazhussain/Barcoda-gym.git
cd Barcoda-gym
```

### 2. تثبيت المتطلبات
```bash
node install.js
# أو يدويا:
cd backend && npm install
cd ../frontend && npm install
```

### 3. إعداد قاعدة البيانات
```bash
# انسخ ملف البيئة
cp backend/.env.example backend/.env

# عدّل البيان��ت حسب جهازك
nano backend/.env
```

### 4. تشغيل التطبيق
```bash
# Linux/Mac
./start.sh

# Windows
start.bat
```

## الـ API الأساسية 🔌

### الأعضاء
- `GET /api/members` - قائمة الأعضاء
- `GET /api/members/:id` - بيانات عضو معين
- `POST /api/members` - إضافة عضو جديد
- `PUT /api/members/:id` - تعديل بيانات عضو
- `DELETE /api/members/:id` - حذف عضو

### الاشتراكات
- `GET /api/subscriptions` - قائمة الاشتراكات
- `GET /api/subscriptions/active` - الاشتراكات النشطة
- `POST /api/subscriptions` - إنشاء اشتراك جديد
- `PUT /api/subscriptions/:id/renew` - تجديد اشتراك
