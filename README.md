# Barcoda Gym - نظام إدارة الصالة الرياضية
# Complete Gym Management System

## 🏋️ نظرة عامة
تطبيق متكامل وشامل لإدارة الصالات الرياضية بجميع أنواعها، يوفر واجهات متعددة (ويب، موبايل، سطح المكتب) مع قاعدة بيانات احترافية ومتطورة.

## ✨ المميزات الرئيسية
✅ **نظام المصادقة والتفويض** - تسجيل الدخول الآمن  
✅ **إدارة المديرين والمشرفين** - إضافة وتعديل وحذف مع الصلاحيات  
✅ **إدارة الأعضاء** - إضافة أعضاء مع الصور والبيانات الكاملة  
✅ **إدارة الاشتراكات** - متابعة فترات الاشتراك والسداد  
✅ **نظام الباركود** - توليد وطباعة باركود لكل عضو  
✅ **قراءة الباركود** - البحث السريع عن الأعضاء  
✅ **التنبيهات الذكية** - تنبيهات تلقائية 30 يوم قبل الانتهاء  
✅ **نظام السداد** - تتبع الفواتير والدفعات  
✅ **التطبيقات المتعددة** - ويب + موبايل + سطح مكتب  
✅ **تغيير اسم الصالة** - تعديل الاسم في أي وقت  

## 📁 البنية الهندسية
```
Barcoda-gym/
├── backend/              # الخادم الخلفي
├── frontend/             # واجهة الويب
├── mobile/               # تطبيق الموبايل
├── desktop/              # تطبيق سطح المكتب
└── database/             # قاعدة البيانات
```

## 🚀 البدء السريع
```bash
cd backend
npm install
npm run dev
```
